Given a list of integers number, write a function to return the largest integer in the list.

You must define the function solve(number).


Example 1:

Input: number = [1, 5, 3, 9, 2]
Output: 9

Example 2:

Input: number = [-10, -5, -2]
Output: -2

Constraints:

  • The list will contain at least one number.
  • Integers can be negative.
