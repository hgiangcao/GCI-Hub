Given a list of integers, return the count of numbers that are divisible by either 3 or 7.

You must define the function solve(number).


Example 1:

Input: number = [3, 7, 21, 4, 5]
Output: 3

Example 2:

Input: number = [1, 2, 4]
Output: 0

Constraints:

  • Zero is divisible by everything, treat 0 as divisible.
