Given a list of integers and integers X and Y, return the first number in the list divisible by both X and Y. Return -1 if none found.

You must define the function solve(number, x, y).


Example 1:

Input: number = [10, 12, 30], x=2, y=3
Output: 12

Example 2:

Input: number = [10, 15], x=7, y=2
Output: -1

Constraints:

  • x and y are non-zero.
