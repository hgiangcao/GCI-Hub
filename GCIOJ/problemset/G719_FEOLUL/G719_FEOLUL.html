Simple Calculation of Two Numbers

Take two numbers from keyboard, a and b as input. Calculate the result of the following expression (and assign to variablle res:

res = (a + b) + (a - b) + (a * b) + (a / b)

Print the result res to the screen.

Example 1:

Input: a =2; b = 3
Screen output: 12.5

Example 2:

Input: a =1; b = 3
Screen output: 5.333333
