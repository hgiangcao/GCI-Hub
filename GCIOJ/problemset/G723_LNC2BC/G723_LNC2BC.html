Get weight (in kg) and height (in centimeter) from keyboard, then

compute BMI based on weight (kg) and height (meter) as follow:

BMI = weight / (height * height)

Note: Remember to convert the height from centimeter to meter

Example 1:

Input: height = 160; weight = 65
Screen output: 25.390625

Example 2:

Input: height = 180; weight = 85
Screen output: 26.23456790123457
